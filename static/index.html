What It Predicts

OncoProfilo predicts PAM50 molecular subtype and a Cox-style survival risk score from RNA-seq FPKM expression. The classifier returns Basal, Her2, LumA, LumB, or Normal-like subtype probabilities.

Input Contract

The trained network receives 2,000 selected gene features in a fixed order. The custom interface maps gene IDs into that order, applies log2(FPKM + 1), scales values with the saved StandardScaler, and fills any missing expected genes with 0.

Use

This build is intended for model demonstration and engineering validation. Real clinical interpretation needs external validation, calibration, and review by qualified medical professionals.
